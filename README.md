# 🧠 MindCare

> Une application bienveillante dédiée au suivi de la santé mentale, au journaling et au bien-être quotidien.

MindCare est une plateforme complète qui aide les utilisateurs à prendre soin de leur santé mentale de manière interactive. L'application propose des outils de suivi de l'humeur, un journal intime, des exercices de respiration, ainsi qu'un assistant conversationnel (chat) doté de fonctionnalités vocales.

---

## ✨ Fonctionnalités Principales

- **🔒 Authentification Sécurisée** : Inscription et connexion gérées de manière sécurisée (intégration Auth0).
- **📝 Suivi de l'humeur & Journaling** : Interface intuitive pour enregistrer ses émotions quotidiennes et tenir un journal.
- **🤖 Assistant Conversationnel (Chat)** : Un espace d'échange intelligent pour guider et écouter l'utilisateur, avec support vocal.
- **🧘 Exercices de Respiration** : Module interactif pour aider à la relaxation et la gestion du stress.
- **📊 Tableau de bord (Suivi)** : Visualisation de l'évolution de l'humeur au fil du temps.
- **🌓 Mode Sombre / Clair** : Interface utilisateur adaptative et accessible, propulsée par Tailwind CSS et shadcn/ui.
- **📱 Responsive Design** : Expérience fluide optimisée pour mobile, tablette et desktop.

---

## 🛠️ Stack Technique

**Frontend**

- [React 18](https://react.dev/) - Bibliothèque UI
- [TypeScript](https://www.typescriptlang.org/) - Typage statique
- [Vite](https://vitejs.dev/) - Bundler ultra-rapide
- [Tailwind CSS](https://tailwindcss.com/) - Framework CSS utilitaire
- [shadcn/ui](https://ui.shadcn.com/) - Composants UI accessibles et personnalisables

**Backend**

- [Node.js](https://nodejs.org/) & [Express](https://expressjs.com/) - Serveur et API REST
- [MongoDB](https://www.mongodb.com/) - Base de données NoSQL (via Mongoose)
- [Auth0](https://auth0.com/) - Gestion des identités et de l'authentification _(en cours d'intégration)_

Assurez-vous d'avoir installé sur votre machine :

- [Node.js](https://nodejs.org/) (version 18 ou supérieure)
